Project Darkstar Request Example
The Project Darkstar Request Example is an example application built on top of the Project Darkstar stack. It is used primarily for performance testing of the Project Darkstar server itself.

See the testing page for performance test results and information about running tests.

Source code
The publicly accessible Subversion repository contains all of the source code for the request project.
Issue tracker
Issuezilla is used to track all issues with the request project including defects, enhancements, and outstanding tasks.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for the request project. For development discussions and general announcements, please join the dev and announce lists for the games-darkstar parent project.
