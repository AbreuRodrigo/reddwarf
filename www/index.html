Project Darkstar Tests
The sgs-tests project hosts integration and performance tests to be run against the Project Darkstar core itself, as well as Project Darkstar applications. It is currently under heavy development and should be considered a work in progress. This site provides access to the source code repository, issue tracker, and mailing lists for this project.
Source code
The publicly accessible Subversion repository contains all of the source code for the sgs-tests project.
Issue tracker
Issuezilla is used to track all issues with the sgs-tests project including defects, enhancements, and outstanding tasks.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for the sgs-tests project. For development discussions and general announcements, please join the dev and announce lists for the games-darkstar parent project.
