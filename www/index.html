Project Darkstar Hack Example
The Project Darkstar Hack Example is an example application built on top of the Project Darkstar stack. The code is released into the Creative Commons Public Domain and is available for unrestricted use.
Source code
The publicly accessible Subversion repository contains all of the source code for the hack project.
Issue tracker
Issuezilla is used to track all of the issues with the hack project including defects, enhancements, and outstanding tasks.
Click here for a full listing of all the open issues for the project.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for the hack project. For development discussions and general announcements, please join the dev and announce lists for the games-darkstar parent project.
