sgs:deploy-dependencies

Full name:

com.projectdarkstar.maven.plugin:sgs-maven-plugin:1.0-alpha-1:deploy-dependencies

Description:

Deploys a jar or jar files into a Project Darkstar server installation.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
sgsHome File - Directory where the Project Darkstar server is installed.

Optional Parameters

Name Type Since Description
deployDir File - The deploy directory of the Project Darkstar installation. Defaults to the "deploy" subdirectory under sgsHome.
excludeArtifactIds String[] - A list of artifactIds from the project's dependencies to exclude from the deployment list. By default, all project dependencies are deployed.
excludeClassifiers String[] - A list of classifiers to exclude.
excludeScopes String[] - A list of scopes to exclude.
excludeTransitive boolean - Exclude transitive dependencies. Default value is false.
Default value is: false.
excludeTypes String[] - A list of types to exclude.
includeArtifactIds String[] - A list of artifactIds from the project's dependencies to deploy into the server installation. By default, all project dependencies are deployed.
includeClassifiers String[] - A list of classifiers to include. By default, all classifiers are included.
includeScopes String[] - A list of scopes to include. By default, all scopes are included.
includeTypes String[] - A list of types to include. By default, all types are included.
unpack boolean - If true, artifacts to deploy will be unpacked into the deploy directory. Otherwise, they will simply be copied. Defaults to "false"
Default value is: false.
