sgs:configure

Full name:

com.projectdarkstar.maven.plugin:sgs-maven-plugin:1.0-alpha-2:configure

Description:

Configures a Project Darkstar server installation by overlaying sgs-boot.properties, sgs-server.properties, and sgs-logging.properties files into the conf directory of the installation.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
sgsHome File 1.0-alpha-1 Directory where the Project Darkstar server is installed.

Optional Parameters

Name Type Since Description
sgsBoot File 1.0-alpha-1 The file used to override the sgs-boot.properties configuration file of the Project Darkstar server.
sgsLogging File 1.0-alpha-1 The file used to override the sgs-logging.properties configuration file of the Project Darkstar server.
sgsServer File 1.0-alpha-1 The file used to override the sgs-server.properties configuration file of the Project Darkstar server.
