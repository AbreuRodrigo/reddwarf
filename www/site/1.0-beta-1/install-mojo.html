sgs:install

Full name:

com.projectdarkstar.maven.plugin:sgs-maven-plugin:1.0-beta-1:install

Description:

Installs a Project Darkstar server

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
artifactId String 1.0-alpha-1 The artifactId of the Project Darkstar server distribution. Default value is sgs-server-dist.
Default value is: sgs-server-dist.
groupId String 1.0-alpha-1 The groupId of the Project Darkstar server distribution. Default value is com.projectdarkstar.server
Default value is: com.projectdarkstar.server.
outputDirectory File 1.0-alpha-1 The location to unpack and install the Project Darkstar server distribution. Default value is ${project.build.directory}
sgsHome File 1.0-alpha-1 Directory where the Project Darkstar server is installed.
version String 1.0-alpha-1 The version of the Project Darkstar server distribution.

Optional Parameters

Name Type Since Description
classifier String 1.0-alpha-1 The classifier of the Project Darkstar server distribution. Default is no classifier
cleanSgsHome boolean 1.0-alpha-2 If true, remove the given sgsHome directory which is presumably the resulting installation after installing. Otherwise, if a previous installation exists, it is preserved, and the install is skipped.
Default value is: false.
type String 1.0-alpha-1 The type of the Project Darkstar server distribution. Default value is zip.
Default value is: zip.
