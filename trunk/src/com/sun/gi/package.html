Package Specification

This package contains the entire Sun Game Server system in a variety of sub directories. The sub directories and their uses are as follows: comm: The Communications layer (teir 1) logic: The Game Logic layer (tier 2) objectstore: The Object Store layer (tier 3) utils: General code utlities required by the other packages. gamespy: Bindings to the gamespy SDK (used by the Gamespy User Manager) This code base has the following external dependancies: JRMS1.1 The TimesTen JDK1.4 JDBC driver
