Package Specification

This package handles the transport of packets of data from source to destination. It has three eprimary end points for a communication: a user, a channel or the virtual server. Users as self-explainatory. channles are groups of users. The virtual server is a single end-point for the entire logic layer. Data sent to thsi end-point is routed internally to the most appropriate logic-engine.
